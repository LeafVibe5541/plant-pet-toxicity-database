## Panoramica

Molti strumenti di identificazione classificano la tossicità per animali solo come "tossica/non tossica". È un modello troppo semplice.

Questo database fornisce dati dettagliati:
- Per organo della pianta (foglia, fusto, polline, linfa, ecc.)
- In base a dose e taglia dell'animale
- In base al meccanismo tossico
- In base al contesto reale di esposizione

## Concetti chiave

- **PAWS (Pet-Aware Warning System)**: valutazione multidimensionale della tossicità, per organo e adattata alla taglia dell'animale
- **PF-BRI (PlantFun Bio-Rhythm Index)**: raccomandazioni dinamiche di irrigazione basate sul VPD (deficit di pressione di vapore)
- **Micro-distance Pathological Algorithm**: distinzione precoce tra stress biotico e abiotico

## Metodologia dei dati

1. Risoluzione a livello di organo
2. Documentazione dei meccanismi tossici
3. Specificità gatto/cane
4. Benchmark basati su evidenze
5. Aggiornamento trimestrale
